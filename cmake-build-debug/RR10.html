Requisito de restricción RR10 Unicidad NIF
DescripciónLa información almacenada por el sistema deberá satisfacer la siguiente restricción: los NIF deberán ser únicos para cada cliente, es decir, no puede haber dos clientes distintos con el mismo NIF.
